# TheraPea 

TheraPea is a comprehensive telehealth web application designed to connect patients with licensed mental health professionals. The platform prioritizes accessible, high-quality care through secure video sessions and features an intelligent Smart Triage System to assess patient needs.

## Key Features

* **Smart Triage Assessment:** An intelligent routing system that evaluates patient risk levels (Low, Medium, High) to recommend the appropriate level of care.
* **Role-Based Access Control:** Distinct workflows and dashboards tailored for Patients, Doctors, and Administrators.
* **Secure Telehealth Infrastructure:** Built to handle secure appointments and future integration for HIPAA-compliant video sessions.
* **Modern, Calming UI:** A therapeutic design system utilizing Sage Green and Soft Lavender palettes to reduce cognitive load and anxiety for users.

## Tech Stack

### Frontend
* **Framework:** React.js (via Vite)
* **Styling:** Tailwind CSS (v3)
* **Routing:** React Router DOM
* **Architecture:** Mobile-first strategy with responsive fluid layouts.

### Backend & Database
* **Framework:** Java Spring Boot (v3.x)
* **Language:** Java 21
* **Database:** PostgreSQL (hosted on Supabase)
* **ORM:** Spring Data JPA / Hibernate
* **Environment Management:** Spring Dotenv
